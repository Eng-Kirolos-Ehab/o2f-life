ربط GitHub

أدخل GitHub Personal Access Token لتفعيل الحفظ والنشر من لوحة التحكم.

للحصول على token: اذهب إلى GitHub Settings → Tokens ← اختر classic ← فعّل صلاحية repo ← انسخ الـ token والصقه هنا.

التوكن يُحفظ في المتصفح فقط ولا يُرسل إلا إلى GitHub API مباشرةً.
الـ token يجب أن يبدأ بـ ghp_ أو github_pat_
